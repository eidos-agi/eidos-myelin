## Claim

Reflexion (Shinn et al., 2023) uses verbal reinforcement learning — agents reflect on task feedback and store reflections in episodic memory buffer. Achieves 91% pass@1 on HumanEval (vs GPT-4's 80%). Key insight from ExpeL follow-up: failure-derived heuristics substantially outperform success-derived ones. Multi-Agent Reflexion (2025) found single-agent reflection has confirmation bias. Difference from myelin: Reflexion is within-trial reflection, myelin is cross-session distillation with human curation gate.
