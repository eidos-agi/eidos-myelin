## Claim

Mem0 (2025) is a scalable memory layer for AI agents — extracts, consolidates, retrieves facts from conversations. 26% uplift over OpenAI's built-in memory, 91% latency reduction via selective retrieval, 90% token reduction. Graph variant (Mem0g) handles relational reasoning. Differs from myelin: Mem0 remembers facts, myelin distills tactics. Mem0 is per-conversation, myelin is cross-session.
