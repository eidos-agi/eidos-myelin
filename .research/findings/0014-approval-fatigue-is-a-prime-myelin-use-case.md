## Claim

Repeated tool approval prompts in Claude Code sessions are a friction pattern that myelin should detect. When a user approves the same tool type 5+ times in a session, or across sessions, that's a learnable pattern: "user trusts file writes in new repos during research phases." This is a behavioral signal that the user model should capture and surface as a recommendation to adjust permission settings or pre-approve tool classes.
