eidos-myelin

Tactical intelligence for Claude Code. Learns what works across sessions. Surfaces it when it matters.

Myelin is the biological sheath that insulates neural pathways, making signals travel faster through repeated use. It doesn't create new neurons — it makes existing pathways faster. Same Claude Code, same tools. Smarter through experience.

01 The Problem

Every Claude Code session starts from scratch. Perfect recall, zero learning.

Lost Knowledge

Patterns vanish between sessions

You figured out that grep --type yaml finds config in 1 call vs 4 with glob. Next session? Gone. You'll waste those same 4 calls again.

Repeated Mistakes

Same errors, different day

Subagents can't write outside the working directory. You discovered this 5 sessions ago. You'll discover it again tomorrow.

Static Instructions

CLAUDE.md doesn't learn

Your instructions file is manually written, never updated, and can't observe its own impact. Research shows auto-generated context files reduce performance by 3%.

Silent Friction

Slowdowns you don't notice

Approving every tool call. Over-reading before writing. Searching the wrong way. These patterns cost minutes per session, hours per month.

02 Three Kinds of Learned Knowledge

Myelin distills session transcripts into three types of actionable intelligence.

⚡

Tactics

Cross-session patterns that consistently work. Promoted after repeated observation, confidence-scored, scoped to project or global.

TACTIC: search-strategy/cockpit-eidos
Confidence: 0.87 (12 sessions)

"grep --type yaml finds config in 1 call. Don't glob first."
⚠

Anti-patterns

Things to stop doing. Distilled from high-effort/low-progress sessions, user corrections, and repeated failures.

ANTI-PATTERN: over-reading
Confidence: 0.95 (corrected 3x)

"Reading >2 files before writing correlates with user frustration."
⚙

User Model

Behavioral profile predicting productive sessions. Not biographical. Transparent, editable, resettable.

USER-MODEL: work-style
From: 47 sessions

"Says 'go' = zero discussion. Prefers proof over description. Checkpoint cadence: ~45 min."

05 Architecture Decision

Three candidates scored against 5 weighted criteria. Clear winner.

Auto-Promote

37
Fast but unsafe. Repeats the AGENTS.md failure mode. Valid V2 path.

Commons + Human Gate

46
Evidence-backed. Human gate is the proven differentiator. Build here.

Standalone Package

31
Over-engineered. Two daemons, two DBs. Not justified at this scale.

AGENTS.md is a notebook.
Myelin is a nervous system.

A notebook can't observe its own impact. It can't learn which instructions help and which hurt. It can't retire stale advice or promote proven patterns. Myelin can — because the evaluation harness feeds back into the distillation pipeline. The tool gets smarter by watching itself work.
